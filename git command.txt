Imp command :
1. Initialize a repository: `git init`
   - Initializes a new Git repository in the current directory.

2. Clone a repository: `git clone <repository-url>`
   - Creates a local copy of a remote repository.

3. Stage changes: `git add <file>`
   - Adds changes in a specific file or directory to the staging area.

4. Commit changes: `git commit -m "commit message"`
   - Records changes in the repository, creating a new commit with a message describing the changes.

5. Push changes: `git push <remote> <branch>`
   - Uploads the local commits to a remote repository.

6. Pull changes: `git pull <remote> <branch>`
   - Fetches changes from a remote repository and merges them into the current branch.

7. Create a new branch: `git branch <branch-name>`
   - Creates a new branch with the specified name.

8. Switch branches: `git checkout <branch-name>`
   - Switches to the specified branch.

9. Merge branches: `git merge <branch>`
   - Integrates changes from the specified branch into the current branch.

10. View branch status: `git branch`
    - Lists all local branches and highlights the current branch.

11. View commit history: `git log`
    - Displays the commit history, including commit messages, authors, dates, and unique identifiers.

12. Discard local changes: `git checkout -- <file>`
    - Discards changes in a specific file, reverting it to the state of the last commit.

13. Resolve conflicts: Manually edit conflicting files and use `git add <file>` to mark conflicts as resolved.

14. Create a new remote branch: `git push -u <remote> <branch-name>`
    - Creates a new branch on the remote repository and sets it as the upstream branch for the local branch.

These are just a few of the many Git commands available. You can find more commands and detailed information in the official Git documentation: https://git-scm.com/doc
